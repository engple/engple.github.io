## 🌟 영어 표현 - program

안녕하세요 👋 오늘은 영어 단어 '**program**'에 대해 알아보려고 해요. '프로그램'이라는 단어는 우리 일상에서 정말 자주 쓰이죠? 영어에서도 '**program**'은 다양한 의미로 사용돼요.

가장 대표적으로는 **컴퓨터 프로그램**을 의미해요. 예를 들어, 우리가 사용하는 앱이나 소프트웨어를 모두 'program'이라고 부를 수 있어요. 또, TV나 라디오에서 방송되는 **방송 프로그램**도 'program'이라고 해요. 그리고 어떤 일을 체계적으로 진행하기 위한 **계획**이나 일정도 'program'이라는 단어로 표현할 수 있어요!

이처럼 'program'은 상황에 따라 여러 가지 뜻으로 활용돼서 정말 유용한 단어예요. 미국식 영어에서는 'program', 영국식 영어에서는 'programme'이라고 쓰기도 하니 참고해 주세요~

## 📖 예문

1. "이 컴퓨터 프로그램은 정말 유용해요."

   "This computer program is really useful."

2. "오늘 밤에 재미있는 TV 프로그램이 있어요."

   "There is an interesting TV program tonight."

3. "여름 방학 동안 특별한 영어 프로그램에 참여했어요."

   "I participated in a special English program during summer vacation."

## 🤝 함께 알아두면 좋은 표현들

### software (소프트웨어)

'software'는 컴퓨터에서 실행되는 모든 프로그램과 관련된 데이터를 의미해요. 'program'과 비슷하게 컴퓨터 작업을 수행하는 명령어들의 집합을 뜻하지만, 좀 더 포괄적인 개념이에요.

- "I installed new software to improve my computer's performance."
- "컴퓨터 성능을 향상시키기 위해 새로운 소프트웨어를 설치했어요."

### application (애플리케이션)

'application'은 특정 작업을 수행하기 위해 만들어진 프로그램을 의미해요. 보통 사용자가 직접 사용하는 프로그램을 가리키며, 'program'의 한 종류로 볼 수 있어요.

- "She uses a photo editing application to enhance her pictures."
- "그녀는 사진을 보정하기 위해 사진 편집 애플리케이션을 사용해요."

### hardware (하드웨어)

'hardware'는 컴퓨터의 물리적인 부품을 의미해요. 'program'과는 반대 개념으로, 소프트웨어가 아닌 컴퓨터 장치 자체를 가리킬 때 사용해요.

- "The hardware of my laptop needs to be upgraded for better performance."
- "더 나은 성능을 위해 내 노트북의 하드웨어를 업그레이드해야 해요."
