컴퓨터를 구성하는 주요 부품들의 영어 표현을 알아볼까요? 오늘은 본체, 그래픽카드, 메모리, 파워서플라이 등 컴퓨터 내부 구성품의 영어 단어와 예문을 소개할게요. 각각의 부품 이름과 발음, 예문까지 함께 익혀보세요!

## 1. 본체 (Case)

컴퓨터의 모든 부품이 들어가는 외부 케이스를 말해요.

### 🗣️ 발음
- <span data-pronunciation="case">발음기호: /keɪs/</span>
- 한국어 발음: 케이스

### 💭 관련 표현
- computer case: 컴퓨터 본체
- tower case: 타워형 본체

### 📝 예문으로 연습하기!

1. "Please open the case to install the new parts."

   "새 부품을 설치하려면 본체를 열어주세요."

2. "My computer case has a glass side panel."

   "제 본체는 옆면이 유리로 되어 있어요."

## 2. 그래픽카드 (Graphics Card)

컴퓨터에서 화면을 출력하는 중요한 부품이에요.

### 🗣️ 발음
- <span data-pronunciation="graphics card">발음기호: /ˈɡræfɪks kɑːrd/</span>
- 한국어 발음: 그래픽스 카드

### 💭 관련 표현
- video card: 비디오 카드
- GPU: 지피유

### 📝 예문으로 연습하기!

1. "I upgraded my graphics card for better gaming."

   "게임을 더 잘하려고 그래픽카드를 업그레이드했어요."

2. "This graphics card supports 4K resolution."

   "이 그래픽카드는 4K 해상도를 지원해요."

## 3. 메모리 (RAM)

컴퓨터가 작업을 처리할 때 임시로 사용하는 기억장치예요.

### 🗣️ 발음
- <span data-pronunciation="ram">발음기호: /ræm/</span>
- 한국어 발음: 램

### 💭 관련 표현
- memory stick: 메모리 스틱
- DDR4 RAM: 디디알포 램

### 📝 예문으로 연습하기!

1. "Adding more RAM makes your computer faster."

   "램을 더 추가하면 컴퓨터가 더 빨라져요."

2. "My computer has 16GB of RAM."

   "제 컴퓨터에는 램이 16기가 있어요."

## 4. 파워서플라이 (Power Supply)

컴퓨터에 전원을 공급해주는 장치예요.

### 🗣️ 발음
- <span data-pronunciation="power supply">발음기호: /ˈpaʊər səˌplaɪ/</span>
- 한국어 발음: 파워 서플라이

### 💭 관련 표현
- power unit: 파워 유닛
- PSU: 피에스유

### 📝 예문으로 연습하기!

1. "The power supply must match your computer’s power needs."

   "파워서플라이는 컴퓨터의 전력 요구에 맞아야 해요."

2. "I replaced my old power supply yesterday."

   "어제 파워서플라이를 새 걸로 바꿨어요."

## 5. 메인보드 (Motherboard)

컴퓨터 부품들이 연결되는 중심 회로판이에요.

### 🗣️ 발음
- <span data-pronunciation="motherboard">발음기호: /ˈmʌðərˌbɔːrd/</span>
- 한국어 발음: 머더보드

### 💭 관련 표현
- [main](/blog/in-english/1404.main/) board: 메인보드
- system board: 시스템 보드

### 📝 예문으로 연습하기!

1. "The motherboard connects all components together."

   "메인보드는 모든 부품을 연결해줘요."

2. "I bought a new motherboard for my PC build."

   "PC 조립하려고 새 메인보드를 샀어요."

## 6. 쿨러 (Cooler)

컴퓨터 부품의 열을 식혀주는 냉각 장치예요.

### 🗣️ 발음
- <span data-pronunciation="cooler">발음기호: /ˈkuːlər/</span>
- 한국어 발음: 쿨러

### 💭 관련 표현
- CPU cooler: CPU 쿨러
- cooling fan: 쿨링팬

### 📝 예문으로 연습하기!

1. "A good cooler keeps your computer from overheating."

   "좋은 쿨러가 있으면 컴퓨터가 과열되지 않아요."

2. "I need to clean the dust off my cooler."

   "쿨러에 낀 먼지를 청소해야 해요."

## 7. SSD (Solid State Drive)

빠른 속도로 데이터를 저장하는 저장장치예요.

### 🗣️ 발음
- <span data-pronunciation="SSD">발음기호: /ˌɛs ɛs ˈdiː/</span>
- 한국어 발음: 에스에스디

### 💭 관련 표현
- solid state drive: 솔리드 스테이트 드라이브
- storage: 저장장치

### 📝 예문으로 연습하기!

1. "Installing an SSD makes your computer boot faster."

   "SSD를 설치하면 컴퓨터가 더 빨리 켜져요."

2. "I store my important files on an SSD."

   "중요한 파일은 SSD에 저장해요."
